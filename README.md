## Next.js × Hasura

Next.js と Hasura を使用した技術ブログ風のポートフォリオです。

※ 無料版 Hasura のため、一ヶ月 1GB の制限を超えると利用できない可能性があります

[Next.js × Hasura](https://nextjs-hasura.tk)

## 技術選定

詳細は、[今回の技術選定](https://nextjs-hasura.tk)をご覧ください。

### フロント

- Next.js
- Typescript
- Recoil
- Apollo Client
- Mui (旧 Material UI)
- SCSS

### サーバー

- Hasura (GraphQL サーバー)
